Fashion Shopping App

Role: UX Research · 설문조사 · 와이어프레임 · UI 디자인 · 프로토타이핑

기간: 2024년 5월 – 6월

📌 Introduction

이 프로젝트는 쇼핑몰 앱을 사용하면서 발생하는 불편함과 혼란을 해결하고, 사용자에게 직관적이고 명확한 UI를 제공하는 것을 목표로 했습니다. 사용자 조사와 반복적인 테스트를 통해 보다 스마트하고 효율적인 쇼핑 경험을 설계했습니다.

🎯 Project Goals

  • 사이즈 정보의 불확실성 최소화로 쇼핑 신뢰도 향상
  • 상품 상세 정보와 리뷰 접근성 개선
  • 쇼핑 중 발생할 수 있는 수량 선택 실수 방지

🧠 Design Thinking Process

Ideate → Empathize → Define → Prototype → Test

🔍 Research & Methodology

구글 폼을 활용한 설문을 통해 총 10명의 쇼핑 앱 사용자 데이터를 수집했습니다. 응답자들은 자신의 쇼핑 빈도, 중요 요소, 불편 사항 등에 대해 구체적으로 응답하였고 이를 통해 중요한 UX 인사이트를 확보했습니다.

  • 쇼핑 빈도 및 자주 사용하는 앱
  • 구매 시 중요하게 생각하는 요소
  • 사이즈 정보 확인 방식
  • 리뷰 확인 시의 불편함

👥 Persona

설문 결과를 바탕으로 다양한 사용자 유형을 정의하였고, 공통적으로는 빠른 정보 확인, 리뷰 필터링, 정확한 사이즈 제공이 필요하다는 점이 도출되었습니다.

💬 Empathy Map

사용자의 감정, 생각, 행동을 시각적으로 정리한 공감 맵을 통해 불편한 지점과 개선 우선순위를 명확히 파악했습니다.

🔀 User Flow

앱 실행 → 소셜 로그인 → 상품 검색 → 필터/정렬 → 리뷰 확인 → 장바구니 추가 → 쿠폰 확인 → 결제

📝 Wireframe

검색, 필터, 상품 상세, 리뷰, 결제까지의 흐름을 반영한 초기 와이어프레임을 제작했고, 피드백을 바탕으로 반복 개선했습니다.

🎨 Color & Typography

🔧 Key Features

1. 로그인 & 홈화면

네이버 및 카카오 소셜 로그인으로 빠른 가입/접속. 홈에서는 할인 배너, 인기 검색어 등을 직관적으로 노출.

2. 검색 및 필터

상품 검색 시 가격, 성별, 할인 여부 등 다양한 조건으로 필터링 가능. 인기순, 최신순 등으로 정렬 가능.

3. 상품 상세 및 사이즈 정보

상세정보 탭에 소재, 색상, 제조사 정보와 함께 실측 사이즈 제공. 불필요한 스크롤 방지를 위해 이미지 지연 로딩 처리.

4. 리뷰

리뷰는 평점/최신/포토/내 체형 기반 정렬 가능. “내 사이즈 리뷰 보기” 기능으로 맞춤형 정보 제공.

5. 추천 및 문의

AI 추천으로 유사 상품 자동 제안. 문의탭에서 배송/반품/교환 관련 질문 확인 가능.

6. 결제

결제 전에 수량/색상/사이즈를 다시 확인. 사용 가능한 쿠폰 정보도 상세히 표시되어 실수 방지.

✅ Conclusion

본 프로젝트는 사용자 조사 기반으로 실질적인 문제를 해결하고, 쇼핑 과정 전반을 고려한 UX/UI 개선을 목표로 진행되었습니다. 명확한 사이즈 정보 제공, 리뷰 개선, 개인화 기능을 통해 보다 직관적이고 만족도 높은 쇼핑 경험을 구현했습니다.
